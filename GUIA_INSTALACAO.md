# Guia de Instalação

## Pré-requisitos
Antes de começar, certifique-se de que você tenha as seguintes ferramentas instaladas:
- [Node.js](https://nodejs.org/) (versão recomendada: 14 ou superior)
- [npm](https://npmjs.com) (geralmente vem com o Node.js)

## 1. Clonando o Repositório
Abra o terminal e execute o seguinte comando para clonar o repositório:

```bash
git clone https://github.com/tiagosouza06/dashboard-faturas.git
```

## 2. Navegando até o Diretório do Projeto
Entre no diretório do projeto com o seguinte comando:

```bash
cd dashboard-faturas
```

## 3. Instalando Dependências
Para instalar as dependências do projeto, execute:

```bash
npm install
```

## 4. Executando o Projeto
Com as dependências instaladas, você pode iniciar o servidor de desenvolvimento:

```bash
npm start
```

Acesse a aplicação em seu navegador em `http://localhost:3000`.
